# Learning Portal using Redux Toolkit 




## Demo

https://learning-portal-redux.netlify.app


## Tech Stack

**Client:** React, React Router, TailwindCSS, Redux Toolkit, RTK Query

**Server:** JSON Server, JSON Server Auth



## Installation

Install with npm using CMD

```bash
  npm install
  npm start
```
